# GoobyDesk-Modern

GoobyDesk Modern is a lightweight but Modern ITSM Platform aimed at Small to Medium sized businesses with less than 200 employees and 10 IT Support Technicians.

**Current Status:** Planning and Pre-Development

## Goals and Requirements

### Technology Stack

- Python3 Flask
- Gunicorn
- Caddy Reverse Proxy
- Ubuntu 26.04 LTS OS
- New Relic monitoring
